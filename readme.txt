An introductory plugin to convert Wordpress Comments to Sugar CRM Cases

== Description ==

An introductory plugin to convert Wordpress Comments to Sugar CRM Cases. This is really a handy plugin when you want to pull the cases from outside Sugar CRM. You can use the comments registered on your Wordpress site to reflect into Sugar CRM.


== Installation ==

1. Upload the full directory into your wp-content/plugins directory.
2. Activate the plugin at the plugin administration page.
3. Open the plugin setting page, which is located under Settings -> Sugar CRM Settings and provide the Sugar CRM URL, and credentials (username/password) and hit the Save option.
4. Now for each comment, there will be an option for "Comment to Case". On clicking of that option, the plugin will send the corressponding comment to Sugar CRM.
